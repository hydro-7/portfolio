# Portfolio 

**Live Demo:** [hydroxyz.vercel.app](https://hydroxyz.vercel.app/)

---

## Tech Stack

Built with:

- Next.js 
- TypeScript
- Tailwind CSS
- Hosted on Vercel

---

## Project Structure

- `src/` – main source code  
- `public/` – static assets  
- `posts/` – blog posts/content (if any)  
- Config files for Tailwind, Next.js, etc.
